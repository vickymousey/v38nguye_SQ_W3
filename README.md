## Side Quest: Week 3

## Setup and Interaction Instructions

To run the sketch locally, open `index.html` in Google Chrome. No additional setup is required.

**Controls:**

- To start the game: Press Enter
- Move left/right: Arrow Keys (player 1) or A/D (player 2)
- To reset the game: Press Enter
- To attack: F (player 1) or K (player 2)
- To block: G (player 1) or L (player 2)

**Opening the Chrome Console**

- **Windows:** Press `F12` or `Ctrl + Shift + J`, then click the **Console** tab
